# HR-Data-Analytics-End-to-End-EDA-on-2M-Employee-Records
This project focuses on end-to-end HR data analytics using Python to extract actionable insights from a large-scale HR dataset containing 2 million employee records. The analysis answers real-world HR and business questions related to employee distribution, salary trends, attrition, performance, hiring patterns, and work modes.


The project demonstrates strong skills in data cleaning, exploratory data analysis (EDA), aggregation, visualization, and business interpretation, making it suitable for Data Analyst / HR Analyst roles.
